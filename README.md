# TerminalX

A custom Android terminal emulator with built-in X11 server support.

## Features
- Native C rendering engine (VT100 + X11)
- Kotlin + Jetpack Compose UI
- Tabbed terminal and X11 sessions
- Full customization via settings
- Lightweight, optimized for older Android devices

## Build
```bash
./gradlew assembleRelease
```
